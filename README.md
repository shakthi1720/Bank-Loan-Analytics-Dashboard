# Bank Loan Analytics Dashboard
## Project Overview
This Power BI dashboard provides insights into bank loan applications, funded amounts, repayments, interest rates, DTI ratios, and loan performance analysis.
The project helps stakeholders monitor lending performance and identify loan trends using interactive visualizations.
---
## Dashboard Features
### Summary Dashboard
- Good Loan vs Bad Loan Analysis
- Funded Amount vs Repayment
- Loan Status Comparison
- Interest Rate Analysis
- DTI Ratio Analysis
### Overview Dashboard
- Monthly Loan Applications Trend
- Loan Applications by State
- Loan Purpose Analysis
- Home Ownership Analysis
- Loan Term Distribution
---
## Key KPIs
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio

## Business Insights
- Majority of loans are categorized as good loans.
- Debt consolidation is the most common loan purpose.
- Fully paid loans contribute the highest repayment amount.
- Mortgage and rent categories dominate home ownership segments.
